# Simple 11-bit R-type Processor
This project is a complete design of an 11-bit R-type processor developed using **Logisim**.

## Technical Specifications
* Instruction Width: 11-bit instructions.
* Storage: 8 D-Flip Flops used for register management.
* Architecture: Dedicated Control Unit and ALU for instruction decoding.
* Tools Used: Logisim.

## Key Insights
* Designed synchronized data paths.
* Implemented fundamental computer architecture concepts: Fetch, Decode, and Execute.
